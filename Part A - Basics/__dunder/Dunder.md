
### `__add__`
```python
a = 1
print(a + 1)
print(a.__add__(1))
```

### `__new__`

Creates and returns and object of a class


### `__init__`

Initializes the object returned by `__new__`

### `__str__`

Printable representation of the class.

### `__eq__`

Compares two objects of the same class.
