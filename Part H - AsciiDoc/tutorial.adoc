// == Footnotes
// A footnote is a note that is placed at the bottom of a page. It is used to explain a term or to
// provide additional information for a concept. Footnotes are referenced using a superscript number
// following the text that the note is attached to. The footnote itself is placed on a line below the
// main text.
//
// This is a footnote reference.[^1]
//
// [^1]: This is a footnote.   // This is a footnote.
//
